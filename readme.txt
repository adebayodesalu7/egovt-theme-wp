A professional WordPress theme for local government portals.

== Description ==

EGovt is a modern, responsive WordPress theme designed specifically for local government and municipal websites. It features a clean, professional design with all the essential components needed for a government portal.

**Key Features:**
- Responsive design that works on all devices
- Custom post types for Events, Council Members, and Documents
- Custom widgets for displaying events, documents, and council members
- Theme Customizer options for easy customization
- SEO-friendly code structure
- Translation-ready
- Cross-browser compatible

**Custom Post Types:**
1. **Events** - Manage city events with date, time, location, and type
2. **Council Members** - Display council member profiles with contact info
3. **Documents** - Share official documents and publications

**Widgets:**
- Recent Events Widget
- Recent Documents Widget
- Council Members Widget

== Installation ==

1. Download the theme ZIP file
2. Go to Appearance > Themes in your WordPress admin
3. Click "Add New" and then "Upload Theme"
4. Choose the ZIP file and click "Install Now"
5. Activate the theme

== Setup ==

**1. Create Pages:**
- Create a page for "Services" (optional)
- Create a page for "Contact" (optional)
- Set your homepage in Settings > Reading

**2. Configure Menus:**
- Go to Appearance > Menus
- Create a new menu and assign it to "Primary Menu" location
- Add your pages to the menu

**3. Customize Theme:**
- Go to Appearance > Customize
- Set your hero section title and subtitle
- Add your contact information
- Upload your logo

**4. Add Content:**
- Add Events: Events > Add New
- Add Council Members: Council > Add New
- Add Documents: Documents > Add New
- Add News: Posts > Add New

== Customization ==

**Theme Options (Customizer):**
- Hero Section: Title, Subtitle, Background Image
- Contact Information: Address, Phone, Email, Office Hours
- Custom Logo

**Custom Fields:**
Events:
- Event Date
- Event Time
- Event Location
- Event Type

Council Members:
- Title
- District
- Email
- Phone
